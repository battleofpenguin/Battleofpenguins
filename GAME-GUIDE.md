# 🎮 Game Guide — Battle of Penguin 3D

## Overview
Battle of Penguin 3D is a browser-based arcade battle game set in an
aurora-lit icy arena. Everything runs client-side from a single HTML file —
no installs, no accounts, no servers.

## Getting Started
1. Open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari).
2. Wait for the title screen to load — you'll see the animated aurora
   background.
3. Start a match from the title screen and follow the on-screen prompts.

## Screens
- **Title Screen** — animated aurora intro, entry point to the game.
- **Battle Screen** — the main gameplay arena.
- (Additional screens/modes may be added over time — see the changelog in
  pull requests.)

## Tips
- The game is fully self-contained in `index.html`; if something looks off,
  try a hard refresh (Ctrl/Cmd + Shift + R) to clear cached assets.
- Best experienced in a desktop browser window at least 1024px wide.
