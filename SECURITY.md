# Security Policy

## Supported Versions

This project is a static, client-side-only HTML game with no backend, no
user accounts, and no data collection. There is no versioned release cycle;
the `main` branch is always the supported version.

| Version | Supported |
| ------- | --------- |
| main    | ✅        |

## Reporting a Vulnerability

If you discover a security issue (for example, an XSS vector if this project
is ever extended with dynamic content, or a supply-chain concern with an
external asset/CDN link), please report it by opening a private security
advisory on GitHub (Security tab → "Report a vulnerability"), or by opening
an issue if the advisory feature is unavailable.

Please include:
- A description of the issue and its potential impact
- Steps to reproduce
- Any relevant browser/environment details

We aim to acknowledge reports within a few days. Since this is a static
front-end-only project, most issues will be low severity, but we take all
reports seriously.
