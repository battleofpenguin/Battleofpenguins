# 🐧❄️ Battle of Penguin 3D

A fast, playable browser game — pure HTML/CSS/JS, no build step, no dependencies.
Open `index.html` and play instantly, or host it for free on GitHub Pages.

## 📌 Why Battle of Penguin 3D?

A self-contained arcade battle game set in a frozen aurora-lit arena — built entirely
in a single HTML file so anyone can clone it, open it, and start playing in seconds.
No npm install, no bundler, no server required.

### ✨ What Sets This Apart

| Feature | Description |
|---|---|
| 🎮 **Zero setup** | Just open `index.html` in a browser — that's it |
| 🏔️ **Single-file game** | All HTML, CSS, and JS bundled into one portable file |
| 🌌 **Animated visuals** | Layered aurora backgrounds, ice glow effects, particle-style animation |
| 📱 **No dependencies** | No frameworks, no build tools — plain web tech only |
| 🤝 **Easy to fork & remix** | Simple structure makes it a great base for your own game jam project |

## 🚀 Quick Start

### Option 1 — Play Online (Recommended)

Enable GitHub Pages for this repo (Settings → Pages → Deploy from `main` branch, root)
and play directly at your Pages URL.

### Option 2 — Run Locally

```bash
# Clone the repository
git clone https://github.com/<your-username>/penguin-battle-3d.git
cd penguin-battle-3d

# Just open it — no build step needed
open index.html      # macOS
start index.html      # Windows
xdg-open index.html   # Linux
```

### Option 3 — Try It Instantly

Drag `index.html` straight into any browser tab.
